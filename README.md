## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v16.x or later
- [Yarn](https://yarnpkg.com/) v3

---

1. このリポジトリをクローンします。
   ([approvers](https://github.com/approvers) のメンバーではない場合は最初に Fork してください)

   ```shell
   git clone https://github.com/approvers/haracho.approvers.dev.git
   ```

2. 依存関係をインストールします。

   ```shell
   yarn
   ```

3. 開発サーバーを起動します。
   ```shell
   yarn dev
   ```
