VC 内の人類に就寝を促すよ。

:::tip

実行者がボイスチャンネルにいないとこのコマンドは安全装置によって封じ込められます。

予約機能についても同様で、実行時間にボイスチャンネルにいないと実行されません。

:::

### `!kaere`

就寝推奨を今すぐ始めます。

### `!kaere bed status`

[強制切断モード](#強制切断モード)の状態を表示します。

### `!kaere bed enable`

[強制切断モード](#強制切断モード)を有効にします。

### `!kaere bed disable`

[強制切断モード](#強制切断モード)を無効にします。

### `!kaere reserve add <時間>`

指定した時間に就寝推奨を予約します。

例: `!kaere reserve add 22:00` で 22:00 に就寝推奨を開始します。

:::caution

同じ時間に予約することは出来ません。

:::

### `!kaere reserve list`

予約状況を表示します。

### `!kaere reserve cancel <時間>`

指定した時間の予約をキャンセルします。

### 強制切断モード

強制切断モードは、就寝推奨が終了後、ボイスチャンネルにいるメンバーを強制的に切断します。
