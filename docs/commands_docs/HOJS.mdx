# †HARACHO ONLINE JUDGING SYSTEM†

<VersionBadge>v1.0.0</VersionBadge>

プログラムが適格かどうかを判定するためのオンラインジャッジシステムです。

### `!jd(!judge) [テスト数=5] [判定結果=AC] [全失敗=false]`

- `[テスト数]` : 数値
  - 判定のアニメーションに使うテストケースの数
  - デフォルトは `5`
  - 最大数は `64`
- `[判定結果]` : 文字列
  - アニメーション終了後の判定
  - デフォルトは `AC`
- `[全失敗]` : Boolean
  - 全てのテストケースを失敗させるかどうか
  - デフォルトは `false`
