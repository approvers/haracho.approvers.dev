# ロール情報表示

<VersionBadge>v1.21.0</VersionBadge>

## `!roleinfo(role) [id]`

指定したロールの情報を表示します。

メンションでは動作しません。

| 項目         | 説明                                                                        |
| ------------ | --------------------------------------------------------------------------- |
| ID           | ロールの ID                                                                 |
| 所属人数     | そのロールを持つメンバーの数                                                |
| ポジション   | ロールリスト内での位置 <br/> (Discord API 上では、最終尾から数えるため注意) |
| カラーコード | そのロールのカラーコード                                                    |
| 作成日時     | そのロールが作成された日時と経過時間                                        |
