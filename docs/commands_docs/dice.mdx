# ダイスロール

<VersionBadge>v1.12.0</VersionBadge>

### `!dice(d) [ダイスロール設定=1d100] [詳細モード=simple]`

- `<ダイスロール設定>`: どのダイスロールを降るかを設定します。
  - 6 面ダイス 2 個であれば `!dice 2d6` (または `!dice 2D6`) と入力します。
- `<詳細モード>`: ダイスロールの出目を表示するかどうかを設定します。
  - デフォルトでは省略します。
- 補足: `!dice`の場合は`!dice 1d100`と同様の入力とみなされます。

| 詳細モード     | 表示形式 |
| -------------- | -------- |
| `s`, `simple`  | 省略表示 |
| `v`, `verbose` | 詳細表示 |
