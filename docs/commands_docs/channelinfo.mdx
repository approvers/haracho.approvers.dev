# チャンネル情報表示

<CommandArgs
    versionAvailableFrom="v1.37.0"
    commandName="channelinfo/channel/chinfo"
    args={[
        {
            name: 'チャンネルID',
            about: '情報を表示したいチャンネルのIDです。チャンネルのメンションを指定しないようにしてください。',
        },
    ]}
/>

指定したロールについて、以下の情報を表示します。

| 項目         | 説明                                                                        |
| ------------ | --------------------------------------------------------------------------- |
| チャンネル名 | チャンネルの名前 |
| チャンネルタイプ | チャンネルのタイプ |
| 管理可能か | クライアントユーザーがチャンネル設定を変更可能か |
| 表示可能か | クライアントユーザー側からこのチャンネルが表示できるか |
