# あっぱれおじゃる様ミーム構文

<VersionBadge>v1.30.0</VersionBadge>

## `!ojaru [arg]`

### Example

`!ojaru 課題の出し忘れで留年`

```
>> あっぱれおじゃる様！見事課題の出し忘れで留年されました！
```

## `!ojaru -g [arg]`

デンボの口調の変更を行います。

### Example

`!ojaru -g 課題の出し忘れで留年`

```
>> あっぱれおじゃる様！課題の出し忘れで留年でございます！
```
