# こるくの玉音放送

  className="badge badge--warning"

>
  ffmpeg必須機能

---

VC 内にこるくの玉音放送を再生するよ。

:::tip

実行者がボイスチャンネルにいないとこのコマンドは安全装置によって封じ込められます。

予約機能についても同様で、実行時間にボイスチャンネルにいないと実行されません。

:::

### `!gyokuon`

こるくの玉音放送を再生するよ。

:::tip ヒント: 玉音放送の元ファイルってどこにあるの?

玉音放送の元ファイルは 限界開発鯖 Google Drive の `colk/` フォルダにあります。

はらちょ Ver はとくにカット等はしていません。

:::
